[![Java](https://img.shields.io/badge/Java-11+-orange.svg)](https://adoptium.net/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 🌟 Features

- 🤖 **AI-Powered Chat** - Uses Google Gemini AI for human-like conversations
- 💬 **Group Support** - Works in groups with @mentions
- 🧠 **Context Memory** - Remembers chat history
- 🎨 **Colored Buttons** - Visually appealing buttons with color indicators
- 👤 **Owner Button** - Direct contact to bot owner
- 🔒 **Secure** - Environment variables for sensitive data

## 📋 Prerequisites

- Java 11 or higher
- Maven
- Telegram Bot Token (from [@BotFather](https://t.me/BotFather))
- Google Gemini API Key ([Get it here](https://makersuite.google.com/app/apikey))
